# JS Camp from Midudev

## With this project I pretend to show all the skills learned from Midudev bootcamp [Jscamp]((https://www.jscamp.dev/javascript/como-recuperar-datos-de-los-elementos-html))

## 📚 Bootcamp content

- **00** - HTML & CSS (`00-html-css`)
- **01** - JavaScript (`01-javascript`)
- **02** - React (`02-react` & `02-react-cdn-version`)
- **03** - Estado Global y React Router / Zustand (`03-react-router-and-zustand`)
- **04** - Node.js & Express (`04-Node` & `04-Express`)
- **05** - Testing (`05-Testing`)
- **06** - TypeScript (`06-Typescript`)
- **07** - Artificial Intelligence (`07-Artificial-Intelligence`)
- **08** - SQL (`08-sql`)
- **09** - CI/CD (`09-ci-cd`)
- **10** - Docker (`10-docker`)

## 🚀 How to begin coding

Every module has real practice challenges where I put all the skills I've learned in practice. Navigate to the corresponding folder and follow the instructions.

```bash
# Clone the repo
git clone git@github.com:AlexanderEstebanZapata1994/jscamp-ci-cd.git

# Navigate to the desired module
cd jscamp-ci-cd/00-html-css

# Start coding!
```

## React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Babel](https://babeljs.io/) (or [oxc](https://oxc.rs) when used in [rolldown-vite](https://vite.dev/guide/rolldown)) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh

## React Compiler

The React Compiler is currently not compatible with SWC. See [this issue](https://github.com/vitejs/vite-plugin-react/issues/428) for tracking the progress.

cd 02-react
